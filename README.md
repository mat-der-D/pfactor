# pfactor

4桁の数を素因数分解するタイムチャレンジ用 Web ツール。  
PC上でペンタブを使い、手書き計算の様子を YouTube Shorts 向けに録画することを想定しています。

**[https://pfactor.smooth-pudding.net](https://pfactor.smooth-pudding.net)**

## キーボードショートカット

| キー | 動作 |
|---|---|
| `Space` | タイマー スタート / 一時停止 |
| `R` | リセット（一時停止中のみ） |
| `Shift` | ペン / 消しゴム 切り替え |

## 動作環境

- モダンブラウザ（Chrome 推奨）
- ペンタブ推奨（pointer イベント対応）

## 録画時の運用

ブラウザをフルスクリーンにしてからページを更新すると、画面サイズに合わせた 9:16 レイアウトで表示されます。
