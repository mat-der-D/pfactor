# 素因数分解チャレンジ アプリ仕様書

## 概要

4桁の数（10面ダイス×4の乱数）を見て素因数分解するタイムチャレンジアプリ。  
PC上でペンタブを使い、手書き計算の様子をYouTube Shorts向けに録画することを想定する。

---

## 技術構成

- HTML + CSS + JavaScript（バニラ）
- 外部依存なし
- アイコン：インラインSVG（Feather Icons準拠）

---

## 画面構成

### 全体レイアウト

- 起動時に `window.innerHeight` を基準に9:16の最大サイズを計算し、コンテナサイズを決定
- サイズはページロード時に一度だけ計算する（リサイズ追従なし）
- コンテナを画面中央に配置
- 上部：操作パネル
- 下部：キャンバスエリア

```js
const h = Math.floor(window.innerHeight * 0.95 / 16) * 16;
const w = h / 16 * 9;
```

- 録画時はブラウザをフルスクリーンにしてからページを更新する運用を想定

## 操作パネル

### 数字欄

- 4つの独立した枠を横並びに表示
- 各枠は0〜9のいずれか1桁を表示
- 待機状態（①）では空欄表示

### タイマー

- 表示形式：`m'ss''mmm`（例：`1'23''456`）
- 待機状態では `0'00''000` を表示
- 精度：ミリ秒

### ボタン

状態ごとのボタン構成：

| 状態 | 左ボタン | 右ボタン |
|---|---|---|
| ①待機 | ▶ active | ⏹ inactive |
| ②カウントアップ中 | ⏸ active | ⏹ inactive |
| ③一時停止中 | ▶ active | ⏹ active |

---

## 状態遷移

```
① 待機
  └─ ▶ 押下 → 数字を抽選 → タイマースタート → ② カウントアップ中

② カウントアップ中
  └─ ⏸ 押下 → タイマー一時停止 → ③ 一時停止中

③ 一時停止中
  ├─ ▶ 押下 → タイマー再開（数字・経過時間を維持） → ②
  └─ ⏹ 押下 → 数字・タイマーをリセット → ①
```

- ② → ① の直接遷移は存在しない（⏹ は ② で inactive）
- ③ → ② 再開時は同じ数字・同じ経過時間を維持する
- キャンバスは状態遷移に連動してリセットしない（手動操作のみ）

---

## 数字の抽選

- ① → ② 遷移時（▶ 押下時）に実行
- 各桁を独立して `Math.floor(Math.random() * 10)` で決定
- 4桁すべてが0になる（`0000`）ケースは再抽選しない（素因数分解の対象外だが稀なので許容）

---

## キャンバス

### ツール

| ツール | 種別 | 動作 |
|---|---|---|
| 鉛筆 | モード切替 | ドラッグで線を描画 |
| 消しゴム | モード切替 | ドラッグで線を消去 |
| ゴミ箱 | アクション | クリックで確認ダイアログ → 全消去 |

- 起動時のデフォルトは鉛筆モード
- 確認ダイアログはブラウザ標準の `confirm()` を使用

### 描画仕様

- 色：`#1a1a1a`（固定）
- 太さ：鉛筆 `3px`、消しゴム `20px`（仮値、動作確認後に調整）
- `lineCap`：`round`
- `lineJoin`：`round`
- タッチイベント対応（ペンタブ使用のため `pointer` イベントを推奨）
